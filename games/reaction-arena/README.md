# reaction-arena

Fri3d Badge 2024 mini-game collection focused on reaction speed.

## Planned MVP

- 3 game modes:
  - Single-button reaction test
  - Direction challenge (joystick/button prompt)
  - Sequence memory sprint
- Local high scores (SD-backed if available)
- Simple text/console UI first, LVGL UI second

## Structure

- `app/main.py` - game entrypoint (to implement)
- `app/` - board adapter + game modules
